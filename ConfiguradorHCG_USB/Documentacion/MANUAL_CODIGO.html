1Introducción

ConfiguradorHCG es un script de automatización desarrollado en PowerShell para la configuración estandarizada de equipos de cómputo del Hospital Civil de Guadalajara. Este sistema permite la instalación desatendida de software, configuración de usuarios, registro en inventario y sincronización con sistemas hospitalarios.

Propósito

Automatizar y estandarizar el proceso de configuración de equipos nuevos, reduciendo el tiempo de preparación de aproximadamente 4 horas manuales a menos de 30 minutos de proceso automatizado.

Requisitos del Sistema

Componente Requisito
Sistema Operativo Windows 10/11 Pro (64-bit)
PowerShell Versión 5.1 o superior
Privilegios Administrador (se auto-eleva)
Conectividad Red del hospital (acceso a servidores 10.2.1.13 y 10.2.1.17)
Internet Requerido para registro en Google Sheets
Nota Técnica El script utiliza auto-elevación de privilegios mediante Start-Process con el verbo RunAs, lo que garantiza que siempre se ejecute con permisos de administrador independientemente de cómo se invoque.

2Arquitectura del Sistema

Auto-Elevación de Privilegios

El script inicia verificando si tiene privilegios de administrador. De no ser así, se reinicia automáticamente con elevación:

# Verificación de privilegios administrativos
if (-not ([Security.Principal.WindowsPrincipal][Security.Principal.WindowsIdentity]::GetCurrent()).IsInRole(
    [Security.Principal.WindowsBuiltInRole]::Administrator)) {

    # Re-ejecutar con privilegios elevados
    Start-Process powershell.exe "-ExecutionPolicy Bypass -File `"$PSCommandPath`"" -Verb RunAs
    exit
}

Identificadores de Seguridad (SID)

Para compatibilidad con Windows en cualquier idioma, se utilizan SIDs en lugar de nombres de grupo:

# S-1-5-32-544 = BUILTIN\Administrators
$GrupoAdmin = (New-Object System.Security.Principal.SecurityIdentifier("S-1-5-32-544"))
    .Translate([System.Security.Principal.NTAccount]).Value.Split('\')[-1]

# S-1-5-32-545 = BUILTIN\Users
$GrupoUsuarios = (New-Object System.Security.Principal.SecurityIdentifier("S-1-5-32-545"))
    .Translate([System.Security.Principal.NTAccount]).Value.Split('\')[-1]
Ventaja de usar SIDs Los Security Identifiers (SID) son constantes universales en Windows. Mientras que el grupo "Administrators" se llama "Administradores" en español, el SID S-1-5-32-544 siempre identifica al mismo grupo, garantizando compatibilidad internacional.

Parámetros

ParámetroTipoDescripción

Comportamiento

  • Reproduce la siguiente nota de Pegasus Fantasy en mensajes OK
function Write-Log {
    param([string]$Mensaje, [string]$Tipo = "INFO")

    $Icon = switch ($Tipo) {
        "OK"    { [char]0x2605 }  # ★ Estrella
        "ERROR" { [char]0x2716 }  # ✖ Cruz
        "WARN"  { [char]0x26A0 }  # ⚠ Alerta

    }

6Sistema de Audio - Pegasus Fantasy

El script integra la melodía del opening de Saint Seiya como sistema de retroalimentación auditiva. Cada paso completado exitosamente reproduce la siguiente nota de la canción.

Estructura de Notas

Las notas se almacenan como un array de hashtables con frecuencia (Hz) y duración (ms):

$Script:PegasusNotas = @(
    @{f=659; d=150},  # E5 - Mi
    @{f=659; d=150},  # E5 - Mi
    @{f=587; d=150},  # D5 - Re
    @{f=659; d=250},  # E5 - Mi
    @{f=784; d=150},  # G5 - Sol
    ... (22 notas en total)
)

Play-PegasusNote

Audio

Propósito: Reproduce la siguiente nota de la melodía Pegasus Fantasy.

Comportamiento

  • Usa el índice $Script:PegasusIndex para trackear la posición
  • Incrementa el índice después de cada reproducción
  • Reinicia al inicio cuando completa todas las notas
  • Utiliza [Console]::Beep(frecuencia, duracion)
function Play-PegasusNote {
    if ($Script:PegasusIndex -lt $Script:PegasusNotas.Count) {
        $nota = $Script:PegasusNotas[$Script:PegasusIndex]
        [Console]::Beep($nota.f, $nota.d)
        $Script:PegasusIndex++
    }
}
Experiencia de Usuario Conforme avanza la configuración, el técnico escucha la melodía construyéndose nota por nota. Al completar todos los pasos, se habrá reproducido la melodía completa de Pegasus Fantasy, proporcionando una confirmación auditiva distintiva del éxito de la operación.
